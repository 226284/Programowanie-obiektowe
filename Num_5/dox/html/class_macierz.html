Detailed Description

template<typename Typ>
class Macierz< Typ >

Klasa Macierz określa macierz znaków składającą się z Wektorów.

W klasie macierz znajdują się takie atrybuty jak: Rozmiar - odpowiada za przechowywanie aktualnego rozmiaru macierzy oraz K[ROZMIAR] - odpowiada za przechowywanie danych w macierzy
